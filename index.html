1. css 盒模型

盒模型由元素内容 内边距 边框 外边距组成 , 盒模型种类有ie盒模型以及标准盒模型

  1. box-sizing: content-box (标准盒模型)

    width = 元素的内容区域宽度

  2. box-sizing: border-box (ie盒模型)

    width = 元素的内容区域+ padding + border

2. margin 塌陷 合并

1. margin 塌陷

原理:父子嵌套元素在垂直方向的margin,父子元素是结合在一起的,他们两个的margin会取其中最大的值. 正常情况下,父级元素应该相对浏览器进行定位,子级相对父级定位. 但由于margin的塌陷,父级相对浏览器定位.而子级没有相对父级定位,子级相对父级,就像坍塌了一样.
在文档流中，父元素的高度默认是被子元素撑开的也就是说 子元素有多高，父元素就有多高,但是当子元素设置浮动之后，子元素会完全脱离文档流此时将会导致子元素无法撑开父元素的高度，导致父元素高度塌陷
解决办法:
1. 给父元素设置变框或内边距
2. 触发bfc(块级格式化上下文) ,改变父级渲染规则 给父盒子添加
1) postion: absolute/fied
2) display: inline-block
3) float: left /right
4) overflow:hidden
以上四种方法都能触发bfc

2. margin合并

原理: 两个兄弟结构元素垂直方向的margin是合并的
解决办法: 同样可以采用bfc方式 实际使用时值通过设置一个margin 来解决距离问题

重绘 重排

浏览器运行机制

  1. 构建dom 树 解析html 文档 将标签转换成 dom树种的DOM node, 生成DOM tree

  2. 解析渲染树 解析对应的css 样式文件(包括js生成的) 构建render tree , render tree 中不包含隐藏的节点 head节点

  3. 布局渲染树 从根节点递归调用 计算每个节点大小 位置 给出每个节点在屏幕上的精确坐标

  4. 绘制渲染树 遍历渲染树 使用ui层绘制每个节点

重绘

当盒子位置 大小及其他属性(字体 颜色…) 确定以后 ,浏览器会吧这些属性按各自特性绘制一遍 将内容呈现在页面上
重绘 是一个元素外观改变所触发的浏览器行为 浏览器会根据元素的新属性重新绘制, 使元素呈现性的外观 ()

重绘的触发条件 (background-color color…)

回流(重排/重构)

当渲染树种的元素规模尺寸 布局 等改变页面需要重新构建 这就称为回流 每个页面第一次加载必定回流

触发回流条件 (任何页面布局和几何属性的改变都会触发回流(重排)
1. 页面渲染初始化
2. 添加或者删除可见元素
3. 元素位置改变,或者使用动画
4. 元素尺寸改变(resize事件改变)
5. 填充的内容改变 比如文本改变 或者图片改变 引起计算值宽高改变
6. 读取某些元素属性(offsetLeft/Height/Width …)

3.nodejs

自己简单实现

const target = "https://cn.bing.com"; // 代理地址  localhost/api/
const express = require("express");
const axios = require("axios");
const app = express();
const bodyParser = require("body-parser"); // 解析post请求参数
app.use(bodyParser.urlencoded({ extended: false }));
//设置跨域访问
app.all("*", function (req, res, next) {
  //设置允许跨域的域名，*代表允许任意域名跨域
  res.header("Access-Control-Allow-Origin", "*");
  //允许的header类型
  //   res.header("Access-Control-Allow-Headers", "content-type");
  //跨域允许的请求方式
  res.header("Access-Control-Allow-Methods", "DELETE,PUT,POST,GET,OPTIONS");
  if (req.method.toLowerCase() == "options") res.send(200);
  //让options尝试请求快速结束
  else next();
});
app.use((req, res, next) => {
  const { body, method, headers } = req;
  // 判断 如果是localhost/api 开头的则代理
  let reg = /^\/api\/.*/;
  if (reg.test(req.url)) {
    let targetUlr = "https://cn.bing.com";
    const url = target + req.url.match(/(?<=\/api).*/);
    console.log(url);
    req.url = url;
    axios({
      url,
      method,
      headers: {
        ...headers,
        host: target.match(/(?<=^(https|http)\:\/\/).*/)[0],
      },
      data: Object.keys(body).length ? body : "",
    })
      .then((result) => {
        //   console.log(result.data);
        //   res.status(200).send(JSON.stringify(result.data))
        res.json(result.data);
      })
      .catch((err) => {
        console.log(err);
        res.status(400).json(err);
      });
  } else {
    next();
  }
});
app.listen(80, (_) => {
  console.log("服务启动,端口号80");
});
